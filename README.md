# RadioKorea Rent Scraper

A premium web scraping tool to search and filter rental listings from RadioKorea.

## Features
- **Multi-page Scraping**: Fetch data from multiple pages simultaneously.
- **Advanced Filtering**: Exclude keywords from Title, Writer, and Category.
- **LocalStorage Persistence**: Exclusion keywords are saved automatically.
- **Clean UI**: Separate columns for Writer, Category, and Location.
- **Stable Scraping**: Built-in 1s + random delay to respect server limits.

## Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Steps
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the application:
   ```bash
   npm start
   ```
   (This runs both the frontend on `localhost:5173` and the backend on `localhost:3001`)
